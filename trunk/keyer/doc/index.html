Morse Keyer 1.1 (20-May-2010)

Morse Keyer is a software keyer that can output CW radio tones, old-fashioned telegraph sounder sounds, or drive a device via a serial port (a physical telegraph sounder or ham rig). This document will hopefully be all you need to get going.

Hover the mouse over the controls on the window to get hints on what they do.

If the program crashes when trying to make sounds, you probably can't use it as it requires Microsoft DirectX.

Keying Inputs

The keyer can be keyed with left and right mouse buttons and/or an iambic keyer attached to a serial port (via the CTS and DSR signals).

Keyer Mode

The keyer has two straight-key and iambic ("squeeze") paddles. The straight key mode is simple, it produces sound in response to key or paddle presses. Either paddle will work.

The iambic mode is normally the "B" mode, which adds a trailing element if the paddle is released "early". Most people prefer this mode. Mode "A" can be selected if you're used to it. For more info on iambic keying, see Using an Iambic Paddle by Chuck Adams, K7QO and What's all this iambic keyer mode A and B stuff, anyhow? The code speed control uses the so-called "PARIS" (or "MORSE") method to establish dit and dah lengths.

Connecting a Paddle via a Serial Port

See the directions at Les Kerr's MorseKOB Tutorial Appendix B or at the end of the CW Communicator manual.

Sound Output

You can select to have CW radio tone or telegraph sounder output, or have the keyer drive an external device such as a telegraph sounder or ham rig. If you select CW Radio Tone, clicking the up and down arrow in the Tone Freq. box will play a short tone at the new frequency. If you select Telegraph Sounder, you can cycle through the available sounder sounds in the same way.

At speeds above 30 wpm, the telegraph sounds can start to sound mushy. This is a limitation in the speed of the sound playing in Windows.

Connecting an External Telegraph Sounder

First you need to make up an interface circuit which uses the serial port's Request to Send (RTS) pin to control the current loop that drives the sounder. You can find out more at Les Kerr's MorseKOB tutorial Appendix D, and at the end of the CW Communicator manual. Once you have the sounder interface connected, in RSS Morse, select your COM port and click the little blank button to the right of the Use check box. Your sounder should click/clack 4 times. If this works, you're all set. Check the Use box, and you'll see the sound output items become disabled. Morse will be sent only to your external sounder.
